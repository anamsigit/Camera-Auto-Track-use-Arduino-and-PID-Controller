# Camera-Auto-Track-use-Arduino-and-PID-Controller
Tracking an object by servos, opencv implemented to PID controller with a distance bewteen center and frame and current object position as input

Software include:
- Python
- OpenCV
- simple_pid

Hardware include:
- Camera
- Computer
- Arduino
- Servo
- Pan-tilt servo
